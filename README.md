# VM-tipping 2026 ⚽

Tippesystem for FIFA World Cup 2026 – bygget med React og Firebase.

## Oppsett lokalt

```bash
npm install
npm start
```

## Teknologi
- React 18
- Firebase Firestore (delt database i sanntid)
- WebRTC (videochat peer-to-peer)
- football-data.org API (live kampresultater)

## Live-resultater
Bytt ut `YOUR_KEY_HERE` i `src/constants.js` med din gratis nøkkel fra [football-data.org](https://www.football-data.org).

## Innlogging
- **Admin:** brukernavn `admin`, passord `ADMIN2026`
- **Deltakere:** registrer med invitasjonskode `VM2026`
